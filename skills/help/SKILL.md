# bb — Available Commands

| Command | What it does |
|---------|-------------|
| `/bb:auto` | Full autonomous mode — stop asking, start doing |
| `/bb:commit` | Commit only your files, conventional format, rebase + push |
| `/bb:clean-tree` | Organise a messy working tree into logical commits |
| `/bb:preflight` | Run type-check, lint, build — report issues with fix options |
| `/bb:pr` | Create a pull request from branch commits |
| `/bb:investigate` | Diagnose bugs across multiple services/packages |
| `/bb:help` | This list |
